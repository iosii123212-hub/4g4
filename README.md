#4g4

**4g4 - V1.0 תוכנת חילוץ וכיווץ קבצים שפיתחתי, שמכווצת יותר קטן מ zip-7**
___
 אם תשאלו אותי איך זה יכול היות שזה עוקף את zip-7 אני יגיד לא יודע כי באמת אין לי מושג, אבל מציאות שזה עוקף! נתתי לתוכנה שפיתחתי לכווץ קבצים, והיא כיווצה יותר קטן!  (גם שהפעלתי את zip-7 על רמה 9..) והתוכנה שלי הרבה יותר קטנה!
בקובץ טקסט התוכנה שלי כיווצה ב - 56% ו - zip-7 כיווצה ב - 51%, בתמונה התוכנה שלי כיווצה ב  33.7% והתוכנה של zip-7 כיווצה ב 31.2

טוב, אז התוכנה שלנו משתמשת במנועים הבאים:
 - BZip2
 - Deflate
 - LZMA2
