# 4g4
___

**4g4 - V1.0 תוכנת חילוץ וכיווץ קבצים שפיתחתי, שמכווצת יותר קטן מ zip-7**
___
 אם תשאלו אותי איך זה יכול היות שזה עוקף את zip-7 אני יגיד לא יודע כי באמת אין לי מושג, אבל מציאות שזה עוקף! נתתי לתוכנה שפיתחתי לכווץ קבצים, והיא כיווצה יותר קטן!  (גם שהפעלתי את zip-7 על רמה 9..) והתוכנה שלי הרבה יותר קטנה!
בקובץ טקסט התוכנה שלי כיווצה ב - 56% ו - zip-7 כיווצה ב - 51%, בתמונה התוכנה שלי כיווצה ב  33.7% והתוכנה של zip-7 כיווצה ב 31.2

אז התוכנה שלנו משתמשת במנועים הבאים:
 - BZip2
 - Deflate
 - LZMA2
איך זה שהתוכנה שלנו עקפה זה בגלל שהתוכנה של zip-7 היא מוסיפה הרבה זבל לכיווץ כמו תאריכים ודברים שייכים למנוע ועוד.

**התכונות**
 - התוכנה תומכת בכיווץ של לפורמט 4g4 (פורמט חדש שלי שמבדיקה שלי עד היום הוא לא קיים..)
 - התוכנה תומכת בכיווץ ל: zip  / tar / tar.gz / tar.bz2 / tar.xz / gz / xz/ bz2.
 - התוכנה מעדכנת את עצמה כשיש עדכונים חדשים.
 - התוכנה מראה למשתמש כמה כיווץ כל מנוע ובאיזה מנוע היא בחרה.
- התוכנה מוסיפה את עצמה למקש ימני של הסייר קבצים.
 - התוכנה מוסיפה את עצמה לתוכנות המותקנות.
